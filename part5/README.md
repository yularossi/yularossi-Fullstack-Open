# Bloglist Frontend

This project implements Fullstack Open exercises 5.1-5.4.

## Features

- Login form shown when the user is not authenticated
- Persistent login using browser local storage
- Logout clears stored credentials
- Add a new blog post when signed in
- Notifications for successful and failed operations
- Proxy configured for backend API calls to `http://localhost:3001`

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the backend from `part4/backend`:

   ```bash
   npm run dev
   ```

3. Start this frontend in `part5`:

   ```bash
   npm run dev
   ```

4. Open the Vite URL shown in the terminal.
